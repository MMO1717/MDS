# 异常检测规则说明

## 检测类型

本系统实现三类异常检测，均采用规则判断方式。

---

## 一、无人宿舍耗电检测

### 规则
```
如果 occupied == false 且 power > 300W：
    判定为"无人宿舍耗电"
    风险等级：中
```

### 说明
宿舍无人时，正常情况只有少量待机功耗（<300W）。若功率超过 300W，说明可能存在设备未关闭，造成能源浪费。

---

## 二、高功率异常用电检测

### 规则
```
如果 power > 1500W：
    判定为"疑似高功率违规用电"
    风险等级：高

如果 700W ≤ power ≤ 1200W：
    视为正常空调用电，不判定为违规

如果 1200W < power ≤ 1500W：
    用电偏高，风险等级：中（需关注）
```

### 说明
宿舍常见电器功率一般在 1000W 以下。超过 1500W 通常意味着使用了电热锅、热得快等违规大功率设备。空调功率通常在 700-1200W 范围内，不作为违规处理。

---

## 三、楼层过载风险检测

### 规则
```
统计同一楼层所有宿舍的当前功率总和 floor_total_power：
如果 floor_total_power > 8000W：
    该楼层所有宿舍标记为"楼层过载风险"
    风险等级：高
```

### 风险等级划分

| 楼层总功率 | 风险等级 |
|-----------|---------|
| 0 - 6000W | 低 |
| 6000 - 8000W | 中 |
| 8000W 以上 | 高 |

---

## 高峰负载预测规则

### 预测公式
```
预测负载 = 当前负载 + 最近3个时间点的平均增长量
```

### 高峰时段
- 重点关注 18:00 - 23:00 晚间高峰
- 当预测负载超过 8000W 时触发高峰预警
