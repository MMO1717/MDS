# 数据字段说明文档

## 数据来源

本数据为模拟生成的宿舍用电数据，覆盖 A 栋 5 层共 50 间宿舍，时间跨度 7 天，每 30 分钟采集一次。

## 字段定义

| 字段名 | 类型 | 含义 | 示例 |
|--------|------|------|------|
| dorm_id | string | 宿舍编号 | A101 |
| floor | int | 楼层（1-5） | 1 |
| time | string | 采集时间（YYYY-MM-DD HH:MM） | 2026-05-12 00:00 |
| power | float | 当前功率（W） | 850.0 |
| energy | float | 累计能耗（kWh） | 5.6 |
| temperature | float | 环境温度（°C） | 27.5 |
| occupied | bool | 是否有人 | true / false |
| abnormal_status | string | 异常状态 | normal / abnormal |
| abnormal_type | string | 异常类型 | 无人宿舍耗电 / 疑似高功率违规用电 / 楼层过载风险 / 无 |
| risk_level | string | 风险等级 | 低 / 中 / 高 |
| suggestion | string | 系统建议 | 建议关闭无人宿舍用电设备 |

## 宿舍类型说明

| 宿舍类型 | 宿舍编号 | 数据特征 |
|---------|---------|---------|
| 正常宿舍 | 大部分宿舍 | 功率平稳，无异常 |
| 无人耗电宿舍 | A101, A102, A401 | occupied=false 但 power 较高 |
| 高功率异常宿舍 | A201, A202, A501 | power 偶尔超过 1500W |
| 高耗能宿舍 | A301, A302, A303 | 长时间功率偏高 |
| 楼层过载宿舍群 | 第3层整体 | 同层多间宿舍功率偏高，易触发楼层过载 |

## 数据量

- 总记录数：16,800 条
- 宿舍数：50 间
- 时间点数：336 个（7天 × 48个/天）
