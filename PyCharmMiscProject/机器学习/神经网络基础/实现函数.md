# **Ⅰ. 5.1 模型构建（无需自定义）**

在 5.1 中，你看到作者手写了模块、组合层的逻辑，但在 PyTorch 中：

> **模型构建常用的工具全部已经封装，不用自己写 Module 来替代它们。**

### **1. nn.Module（基类）**

**用途：** 定义任意神经网络。
**为什么不需要自己写？**：PyTorch 已经提供。

**使用：**

```python
import torch.nn as nn

class MyNet(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc = nn.Linear(20, 10)

    def forward(self, x):
        return self.fc(x)
```

**典型场景：**

* 构建任意网络
* 写 forward
* 包含 PyTorch 提供的层

---

### **2. nn.Sequential（顺序组合层）**

**用途：** 快速组合一堆层，不用写 forward。
**使用：**

```python
net = nn.Sequential(
    nn.Linear(20, 256),
    nn.ReLU(),
    nn.Linear(256, 10)
)
```

**典型场景：**

* MLP、简单 CNN、全连接网络的搭建
* 你不想手写 forward 时

---

### **3. 已经实现好的基础层（核心）**

你几乎永远不需要自己写 Linear、Conv、Dropout。

| 功能      | PyTorch 现成层                        |
| ------- | ---------------------------------- |
| 全连接层    | `nn.Linear`                        |
| 激活函数    | `nn.ReLU`, `nn.Sigmoid`, `nn.Tanh` |
| 卷积层     | `nn.Conv2d`                        |
| 池化层     | `nn.MaxPool2d`, `nn.AvgPool2d`     |
| Dropout | `nn.Dropout`                       |
| 批量归一化   | `nn.BatchNorm1d`, `nn.BatchNorm2d` |
| Flatten | `nn.Flatten`                       |

**典型用法：**

```python
layer = nn.Linear(784, 256)
act = nn.ReLU()
drop = nn.Dropout(0.5)
```

# **Ⅱ. 5.2 参数管理（无需自定义）**

这些功能全部 PyTorch 内置，不需要你实现任何管理逻辑。

---

### **1. model.parameters()**

**用途：** 获取模型所有可训练参数。

```python
for p in model.parameters():
    print(p.shape)
```

---

### **2. model.named_parameters()**

**用途：** 同时得到参数名和权重。

```python
for name, param in model.named_parameters():
    print(name, param.shape)
```

---

### **3. state_dict(), load_state_dict()**

**用途：** 参数读取/写入（保存模型用）。

```python
torch.save(model.state_dict(), "m.params")
model.load_state_dict(torch.load("m.params"))
```

---

### **4. param.grad**

训练后查看梯度

```python
print(model.fc.weight.grad)
```

---

### **5. 自动梯度与优化器（无需自己写 SGD）**

PyTorch 已经实现：

* `optim.SGD`
* `optim.Adam`
* `optim.RMSprop`
* `optim.Adagrad`

无须自己写 Se⁵gd() 函数！

# **Ⅲ. 5.3 延后初始化（PyTorch 已内置）**

你不需要手动延后初始化，因为 PyTorch 内置 Lazy 层。

### **1. nn.LazyLinear**

推断输入大小，不用写 in_features。

```python
layer = nn.LazyLinear(256)
```

第一次 `layer(X)` 会自动初始化。

---

### **2. nn.LazyConv2d**

无需写 in_channels。

```python
layer = nn.LazyConv2d(32, kernel_size=3)
```

---

# **Ⅳ. 5.4 自定义层（大多数功能已有，不需自写）**

PyTorch 已经提供绝大部分常见层：

| 类别            | PyTorch 已内置                      |
| ------------- | -------------------------------- |
| 线性层           | `nn.Linear`                      |
| 卷积层           | `nn.Conv2d`                      |
| 注意力机制         | `nn.MultiheadAttention`          |
| Transformer   | `nn.TransformerEncoder` 等        |
| 激活函数          | `nn.ReLU`, `nn.GELU`, etc        |
| 池化            | `nn.MaxPool2d`                   |
| Normalizaiton | `nn.BatchNorm1d`, `nn.LayerNorm` |
| Dropout       | `nn.Dropout`                     |
| LSTM/GRU      | `nn.LSTM`, `nn.GRU`              |

你只在以下情况需要自定义层：

* 发明了新操作（如论文层）
* 做特殊张量变换（如 CenteredLayer）
* PyTorch 没有提供的功能

# **Ⅴ. 5.5 读写文件（PyTorch 已实现）**

### **1. 保存张量/对象**

```python
torch.save(obj, 'file.pt')
```

适用于：

* tensor
* list
* dict
* state_dict

---

### **2. 加载张量/对象**

```python
obj = torch.load('file.pt')
```

---

### **3. 保存模型参数**

```python
torch.save(model.state_dict(), 'model.params')
```

---

### **4. 加载模型参数**

```python
model.load_state_dict(torch.load('model.params'))
```

---

### **5. eval/train 模式切换（dropout/bn 自动切换）**

```
model.train()  # 启用 dropout
model.eval()   # 关闭 dropout，BN 使用移动平均
```
