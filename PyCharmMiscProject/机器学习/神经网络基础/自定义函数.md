
# PyTorch 全部函数/类汇总

---

# **1. 模型构建与组合（来自 5.1 层和块）**

| PyTorch API          | 作用        | 使用示例                                               |
| -------------------- | --------- | -------------------------------------------------- |
| `nn.Module`          | 所有网络、层的基类 | `class MyNet(nn.Module): ...`                      |
| `nn.Sequential()`    | 顺序容器组合层   | `net = nn.Sequential(nn.Linear(20,10), nn.ReLU())` |
| `super().__init__()` | 正确初始化父类   | 自定义层必写                                             |
| `forward(self, X)`   | 定义前向传播逻辑  | 模型调用时自动运行                                          |
| `net(X)`             | 执行模型前向传播  | 输出预测值                                              |
| `vars(net)`          | 查看模块结构与参数 | 调试用                                                |

---

# **2. 参数管理（来自 5.2）**

| PyTorch API                   | 作用                 | 用法                                       |
| ----------------------------- | ------------------ | ---------------------------------------- |
| `model.parameters()`          | 获取所有参数（梯度下降用）      | `for p in model.parameters(): ...`       |
| `model.named_parameters()`    | 返回 (name, param) 对 | 参数可视化/调试                                 |
| `model.state_dict()`          | 获取所有参数的字典          | 保存模型                                     |
| `load_state_dict()`           | 加载 state_dict      | `model.load_state_dict(torch.load(...))` |
| `param.data`                  | 参数的实际值（不跟踪梯度）      | `param.data *= 0`                        |
| `param.grad`                  | 参数梯度               | 反向传播后查看                                  |
| `nn.Parameter(tensor)`        | 注册可训练参数            | 自定义层中的 weight、bias                       |
| `param.requires_grad_(False)` | 冻结参数               | 迁移学习常用                                   |
| `torch.no_grad()`             | 禁用梯度计算             | 推理、修改参数                                  |

---

# **3. 延后初始化（来自 5.3）**

PyTorch 的默认层需要指定输入维度，但 PyTorch 专门提供了 **Lazy 系列层**。

| PyTorch API                   | 作用                     | 用法                   |
| ----------------------------- | ---------------------- | -------------------- |
| `nn.LazyLinear(out_features)` | 不需要输入维度，第一次 forward 推断 | `nn.LazyLinear(256)` |
| `nn.LazyConv2d(...)`          | 同上，卷积 Lazy 版           | 替代 Conv2d            |
| `model(X)`                    | 第一次 forward 时初始化参数     | 必须有一次输入              |

---

# **4. 自定义层（来自 5.4）**

| PyTorch API                | 作用         | 用法                                |
| -------------------------- | ---------- | --------------------------------- |
| `class MyLayer(nn.Module)` | 自定义新层      | 自己写 forward                       |
| `nn.Parameter(...)`        | 自定义层的可训练权重 | `self.weight = nn.Parameter(...)` |
| `torch.matmul()`           | 张量矩阵乘法     | `torch.matmul(X, W)`              |
| `F.relu()`                 | 非线性激活函数    | `F.relu(linear)`                  |
| `X.mean()`                 | Tensor 求均值 | 无参数自定义层例子用                        |

---

# **5. 文件读写（来自 5.5）**

### **5.5.1 加载/保存张量**

| PyTorch API                  | 作用               | 示例                       |
| ---------------------------- | ---------------- | ------------------------ |
| `torch.save(tensor, 'file')` | 保存任意对象（张量、列表、字典） | `torch.save(x, 'x.pt')`  |
| `torch.load('file')`         | 读取对象             | `x = torch.load('x.pt')` |

支持保存：

* 张量
* 列表
* 字典
* 模型参数（state_dict）
* 任意 python 对象

---

### **5.5.2 加载/保存模型参数**

| PyTorch API                                     | 作用                        | 示例     |
| ----------------------------------------------- | ------------------------- | ------ |
| `model.state_dict()`                            | 获取所有可学习参数                 | 保存用    |
| `torch.save(model.state_dict(), 'p.params')`    | 保存参数文件                    |        |
| `model.load_state_dict(torch.load('p.params'))` | 恢复参数                      |        |
| `model.eval()`                                  | 设置 eval 模式（禁用 dropout/bn） | 推理前必须用 |

> 注意：PyTorch 不保存模型架构，只保存参数。
