🧠 训练循环执行流程解析

下面是整个训练循环的执行过程，逐步解释每一行的作用。

1️⃣ 取一批数据

for X, y in train_iter:

从 train_iter（训练数据集的迭代器）中取出一批样本。
	•	X：输入图片张量，形状为 (batch_size, 1, 28, 28)
	•	y：对应的真实标签，如 [2, 5, 7, 0, 1, ...]

⸻

2️⃣ 前向传播

y_hat = net(X)

将输入 X 传入模型，计算预测值。
模型执行以下步骤：
	1.	展平图片为 784 维；
	2.	计算隐藏层：X @ W1 + b1；
	3.	经过 ReLU 激活函数；
	4.	输出层：H @ W2 + b2；
	5.	得到每张图片对应的 10 个类别得分。

⸻

3️⃣ 计算损失

l = loss(y_hat, y).mean()

计算预测结果与真实标签之间的误差（交叉熵损失），并取平均。

⸻

4️⃣ 清空梯度

updater.zero_grad()

清除上一次训练留下的梯度缓存，防止梯度累积。

⸻

5️⃣ 反向传播

l.backward()

让 PyTorch 自动计算损失对所有参数的梯度：
	•	∂l/∂W1、∂l/∂b1、∂l/∂W2、∂l/∂b2
这些结果会自动存入每个参数的 .grad 属性中。

⸻

6️⃣ 参数更新

updater.step()

优化器根据梯度更新参数，使模型在下次预测中更接近正确答案。

更新公式：
\text{新参数} = \text{旧参数} - \eta \times \text{梯度}

⸻

7️⃣ 累计损失与准确率

train_loss_sum += l.item()
train_acc_sum += (y_hat.argmax(dim=1) == y).sum().item()
n += y.shape[0]

	•	记录当前 batch 的损失和正确预测的样本数；
	•	argmax(dim=1) 获取每个样本预测的类别索引；
	•	(y_hat.argmax(dim=1) == y) 判断预测是否正确。

⸻

📊 总结：每一轮训练流程

步骤	操作	说明
1	取一批数据	从训练集中读取一批样本
2	前向传播	计算预测结果
3	计算损失	衡量预测与真实的差距
4	清空梯度	避免梯度累积
5	反向传播	自动计算参数梯度
6	参数更新	利用梯度下降法更新参数
7	记录指标	累计损失与准确率

整个循环就是在不断重复“喂数据 → 计算 → 更新参数”的过程。
所有训练数据都遍历一遍，就完成一个 epoch（训练轮次）。
