# 感知机、线性回归与 Softmax 回归的区别总结

---

## 🧩 一、核心区别概览

| 模型             | 任务类型 | 输出形式      | 激活函数        | 损失函数                | 输出含义      |
| -------------- | ---- | --------- | ----------- | ------------------- | --------- |
| **线性回归**       | 回归任务 | 实数        | 无           | 均方误差 (MSE)          | 预测连续值     |
| **感知机**        | 二分类  | -1 或 1    | 阶跃函数 (sign) | 感知机损失（仅错误样本）        | 判断类别（硬分类） |
| **Softmax 回归** | 多分类  | 概率分布（和为1） | Softmax     | 交叉熵 (Cross Entropy) | 每类的概率     |

---

## 🧠 二、数学表达

### 1️⃣ 线性回归（Linear Regression）

$$
\hat{y} = \mathbf{w}^\top \mathbf{x} + b
$$

* 输出连续值。
* 损失函数：
  $$
  L = \frac{1}{2}(\hat{y} - y)^2
  $$

📘 举例：预测房价、气温、身高等。

---

### 2️⃣ 感知机（Perceptron）

$$
\hat{y} = \text{sign}(\mathbf{w}^\top \mathbf{x} + b)
$$

* 输出：1 或 -1。
* 激活函数：
  $$
  \sigma(x) = \begin{cases}
  1, & x > 0 \\
  -1, & \text{otherwise}
  \end{cases}
  $$
* 损失函数：
  $$
  L = -y(\mathbf{w}^\top \mathbf{x} + b), \quad \text{当误分类时才更新}
  $$

📘 理解：感知机是最早的神经网络单元，只能处理线性可分的二分类任务。

---

### 3️⃣ Softmax 回归（Softmax Regression）

$$
o_i = \mathbf{w_i}^\top \mathbf{x} + b_i
$$
$$
\hat{y_i} = \frac{e^{o_i}}{\sum_j e^{o_j}}
$$

* 输出：每个类别的概率 (0~1)。
* 损失函数（交叉熵）：
  $$
  L = -\sum_i y_i \log(\hat{y_i})
  $$

📘 理解：是感知机的“概率版”，可处理多分类任务，如数字识别、情感分类等。

---

## 📊 三、任务和应用场景

| 模型        | 输入   | 输出   | 应用场景            |
| --------- | ---- | ---- | --------------- |
| 线性回归      | 特征向量 | 连续值  | 房价预测、股价预测       |
| 感知机       | 特征向量 | 二值   | 猫狗分类、人脸识别（两类）   |
| Softmax回归 | 特征向量 | 多类概率 | 数字识别、文本分类、多标签分类 |

---

## 💡 四、通俗比喻

| 类比      | 线性回归         | 感知机       | Softmax 回归            |
| ------- | ------------ | --------- | --------------------- |
| 🎯 输出类型 | 连续数（温度计）     | 开关信号（对/错） | 各类概率（投票系统）            |
| 输出含义    | 实际数值         | 二选一       | 概率分布                  |
| 举例      | 预测房价 123.5 万 | 是否及格      | 猫 0.7 / 狗 0.2 / 兔 0.1 |

---

## ⚙️ 五、代码对比（PyTorch示例）

```python
import torch
from torch import nn

# 1️⃣ 线性回归
linear_net = nn.Linear(2, 1)  # 2输入1输出
loss_fn = nn.MSELoss()

# 2️⃣ 感知机（二分类）
perceptron_net = nn.Sequential(
    nn.Linear(2, 1),
    nn.Sigmoid()  # 近似 sign
)
loss_fn = nn.BCELoss()  # 二分类交叉熵

# 3️⃣ Softmax回归（多分类）
softmax_net = nn.Sequential(
    nn.Linear(2, 3),  # 3类
    nn.Softmax(dim=1)
)
loss_fn = nn.CrossEntropyLoss()
```

---

## 🧭 六、关系与演化逻辑

```
线性回归 → 感知机 → Softmax回归
（连续值）   （二分类）    （多分类）
```

> 线性回归输出实数，
> 感知机输出类别，
> Softmax回归输出概率。

---

## 🧮 七、总结口诀

> 线性回归算数值，
> 感知机分两类；
> Softmax出概率，
> 都是线性打底配。

* 感知机公式：
  $$ o = \sigma(\langle w, x \rangle + b) $$
* 二分类输出：-1 或 1。
* 对比：

  * vs. 线性回归 → 输出实数
  * vs. Softmax 回归 → 输出概率

> 本质区别：**输出类型 + 损失函数不同**。
